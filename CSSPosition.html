CSS Positioning

Forms

To resize image, set width or height on image, not the container
To put image to the left of text, CSS on image and float:left

To center one item
1. CONTAINER: full-width, flex
2. ITEM: not full-width, margin:auto
To center two items inside a section, CSS on the CONTAINER, not the items
display:flex;
justify-content: space-evenly

Center Text Horizontally
text-align: center;
justify-content: center;

Center Text Horizontally using flex
display:flex;
justify-content: center;

Center Text Vertically
display:flex;
align-items: center;

To make an element scrollable,
height:100px;
overflow:auto;
