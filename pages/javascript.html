Arrays

    forEach
  • array.forEach((element, index, array) => {function}
  • Performs function for each element
  • No return value
  • Cannot mutate original array - use another temp variable to store info
    map
  • array.map((element, index, array) => {function}
  • Performs function for each element, but returns NEW array with results
  • Cannot mutate original array - use another temp variable to store new array
    filter
  • array.filter((element, index, array) => {function}
  • Performs function for each element, but returns NEW array with original elements if callback function is true.
  • Cannot mutate original array - use another temp variable to store new array
    sort
  • array.sort((a,b) => {function}
  • Performs comparison between a and b.
  • If return is < 0, sort a before b
  • If return is > 0, sort b before a
  • If return === 0, keep original order
  • No return value - mutates original array
    reduce
  • array.reduce((accumulator, elementValue, elementIndex, array) => {function}, initial value)
  • Use to add up all elements in array
  • Returns final accumulator value
    split
  • array.split(separator,limit)
  • Parses string into array using separator character
  • Original string is NOT modified
    join
  • Turns arrays into strings
  • arr.join(separator)
  • Original array is NOT modified
    slice
  • Find elements within an array and returns the string
  • array.slice(start,end)
  • Original array is NOT modified
    splice
  • delete ore replace parts of an array
  • array.splice(start, # of elements, replacement value)
  • Original array is IS modified

Objects
    for-in
  • for (let name in person) {}
    Object.keys
  • Takes an object and returns an array of properties
  • let properties = Object.keys(object)
  • Since it is an array, you can use typical array methods (like forEach).
    Object.values
  • returns iterable array of just the values from an object
    Object.entries
  • returns array of each property/value pair as array

Classes

    Basic syntax
  • Similar to function, except with class wrapper
  • Need constructor inside to create object
  • Example:
    class ClassName {
      constructor(parameter1, parameter2){
        this.parameter1 = parameter1;
        this.parameter2 = parameter2;
      }
    }
    Extend Class
  • Need constructor to create object, just like before
  • super calls constructor from parent class to access parent properties
  • Example:
    class SubClass extends ClassName{
      constructor(parameter1, parameter2, parameter3){
        super(parameter1, parameter2);
        this.parameter3 = parameter3;
      }
    }
