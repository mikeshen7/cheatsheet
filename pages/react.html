React basics

    Creating React app
  • npx create-react-app app-name
  • npx - (node package executor)
  • This installs all the required components for a react app
  • Intialized as git repo automatically
    Starting React app
  • `code .` to start VS Code, like normal
  • npm start
  • This starts live server
  1. Import react
  2. Create class component
  3. Export component
    Other info
  • Babel - converts react code into machine-readable javascript code
  • Need one HTML element in index.html that React references (typically div id="root")
