Attribute Selector

[attribute]

Select all elements that have a specific attribute
Specify the value of the attribute [attribute="value"]
Value begins with characters [attribute^="val"]
Value ends with characters [attribute$="ue"]

Value conatins characters [attribute*="alu"]

p[type] {
   color: red;
}
<div > DIV</div>
<p type="type X">P type X</p>
<p type="type Y">P type X</p>
<div type="type Z"> DIV type Z</div>
DIV

P type X

P type Y

DIV type Z
